ABOUT NODEQUEUE MIGRATE
-----------------------

The Nodequeue Migrate module enables the migrating of Drupal 7 Nodequeue data
into Drupal 8 Entityqueue data.


REQUIREMENTS
------------

 * A Drupal 8 installation
 * Entityqueue module - https://www.drupal.org/project/entityqueue


INSTALLATION
------------

Install as usual:
 * Place the entire directory in the /modules folder of your Drupal installation
 * Navigate to Administer > Extend
 * Check the 'Enabled' box next to the 'Nodequeue migrate'
 * Click the 'Save Configuration' button at the bottom.

For general help regarding installation, visit:
https://www.drupal.org/documentation/install/modules-themes/modules-8
